# Medical-Doc-Automator

医療現場（理学療法・作業療法）における事務作業の負担を軽減するために開発しました。 患者基本情報を入力するだけで、ネットワーク上の共有フォルダ内に患者専用のフォルダを作成し、日付や氏名を反映させた各テンプレートファイルを自動配分します。

主な機能 フォルダ階層の自動構築: 患者名・IDに基づいた専用フォルダおよびサブフォルダ（計画書、実施表、経過記録など）を自動作成。

マルチフォーマット対応: Excelだけでなく、Word（.docx）内の特定キーワード（例：◆氏名◆）を置換して書類を作成。

ファイル間連携: 実施表内の氏名セルから、同フォルダ内の経過記録（Word）へ自動的にハイパーリンクを付与。

算定区分連動: 「脳血管」「廃用」「運動器」などの区分に応じ、Excel上のチェックボックスを自動操作。

データベース転記: 作成した患者情報を病棟別の管理シートへ自動でバックアップ。

導入・使用方法 YOUR_SERVER などの定数部分を、実際の共有サーバのパスに書き換えます。

「原本」フォルダ内に、指定の名称でテンプレート（Excel/Word）を配置します。

「患者情報」シートに必要事項を入力し、実行ボタンを押下します。

技術スタック Language: VBA (Excel/Word Object Library)

Application: Microsoft Excel, Microsoft Word

注意事項 本リポジトリのコードは、公開用に個人名やサーバパスをマスク（抽象化）しています。

ネットワーク環境（UNCパス）での動作を想定していますが、権限設定により MkDir 等でエラーが出る場合は管理者設定を確認してください。
